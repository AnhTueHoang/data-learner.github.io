Project Overview

The goal of this project was to build a simple, interpretable model that can predict house prices using a subset of variables from the Ames Housing dataset. Instead of using every possible feature, I focused on a small group of meaningful square footage features to keep the project clear and beginner-friendly.

This project helped me practice working with real data, understand the modeling process, and start building confidence in my ability to ask questions and find answers using data.

Dataset

  • Source: AmesHousing_simple.csv
  • Target variable: SalePrice (house sale price)
  • Selected features:
    • BsmtFin SF 1 ‚Äì finished basement square footage
    • Bsmt Unf SF ‚Äì unfinished basement square footage
    • Total Bsmt SF ‚Äì total basement area
    • 1st Flr SF ‚Äì first floor square footage
    • 2nd Flr SF ‚Äì second floor square footage
    • Gr Liv Area ‚Äì above-ground living area

Process & Approach

  • Loaded the dataset into a Google Colab notebook using pandas.read_csv().
  • Explored the data with df.head(), df.info(), and df.describe().
  • Checked for missing values and confirmed the data was mostly clean.
  • Visualized relationships between square footage and price using histograms and scatterplots.
  • Selected a small set of interpretable features focused on different types of space in the home.
  • Split the data into training and test sets using train_test_split.
  • Trained a linear regression model to predict SalePrice.
  • Evaluated the model using Mean Squared Error and a predicted vs actual price scatter plot.
  • Interpreted the model coefficients to understand which features mattered most.

Key Findings

  • Above-ground living space (1st and 2nd floor) had the strongest impact on price. The coefficients suggested that each additional square foot on these floors added over \$120 to the predicted sale price.
  • Basement space also increased house value, but less than the main living areas. Finished basement space was more valuable than unfinished space.
  • Multicollinearity appeared between Gr Liv Area and the individual floor features. Because they contain overlapping information, one of the coefficients appeared negative even though more living area clearly increases price. This taught me an important lesson about being careful when including highly correlated features in the same model.
  • Even a simple linear model can provide useful insights about which features matter most, as long as you choose interpretable variables and think carefully about the results.
